Nagoya, Japan - An Alternative Approach to City Zoning

This map, created as my culminating Geovisualization project, was designed to demonstrate the Japanese approach to urban zoning, and the ways in which it encourages walkable, community-focused cities. It was created in ArcGIS Pro using resources from various Japanese governmental agencies and content from Esri Japan and Community Maps.

Kern County, California - Salvation in the Air

This poster is the output of a site-suitability analysis for fog harvesting devices in the drought-ridden Kern County, California. Fog harvesting offers an environmentally friendly approach to drought-mitigation in agriculture. This suitability analysis makes use of a DEM created from Kern County Contour Data.

California - Natural Disaster Index for Homeowners

This script, created with a group as the culminating assignment for the Problem Solving and Programming course, is designed to allow potential home buyers in California to assess the natural disaster risks for properties prior to making a purchase. It operates by using geoprocessing to compute the intersection of user input coordinate points with a set of polygon layers representing three different natural disaster risks.

Script Download

Note: The accompanying geodatabase must be kept within the same folder as the script to run correctly. The csv file is included for demonstration purposes, but any csv formatted in the same way may be used.
